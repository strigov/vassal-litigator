# catalog — Каталогизация документов

Когда пользователь запускает эту команду:

1. Прочитай скилл `skills/catalog/SKILL.md` и следуй его инструкциям.
2. Убедись, что `.vassal/index.yaml` существует и не пуст. Если пуст — предложи сначала `/vassal-litigator:intake`.
3. Выполни полный pipeline: загрузка → обогащение описаний → генерация xlsx → отчёт.
4. Для генерации xlsx используй скилл `xlsx` (если доступен) или `openpyxl` напрямую.
