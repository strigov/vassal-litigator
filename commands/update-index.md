# update-index — Обновление индекса

Когда пользователь запускает эту команду:

1. Прочитай скилл `skills/update-index/SKILL.md` и следуй его инструкциям.
2. Убедись, что `.vassal/index.yaml` существует. Если нет — предложи `/vassal-litigator:init-case`.
3. Выполни полный pipeline: сканирование → сравнение → preview → apply → верификация.
