# update-index — Обновление индекса

Когда пользователь запускает эту команду:

1. Убедись что `.vassal/index.yaml` существует. Если нет — предложи `/vassal-litigator-cc:init-case` и остановись.
2. Прочитай `skills/update-index/SKILL.md`. Следуй Фазе 1 (Сканирование и Preview, read-only).
3. Покажи результат сканирования Сюзерену. Жди подтверждения действий.
4. После подтверждения: продолжай по `skills/update-index/SKILL.md` к apply-фазе.
5. После завершения apply: следуй verify-фазе и покажи Сюзерену резюме.
