# add-opponent -- Прием документов оппонента

Когда пользователь запускает эту команду с "$ARGUMENTS":

1. Прочитай скилл `skills/add-opponent/SKILL.md` и следуй его инструкциям.
2. Если `$ARGUMENTS` содержит пояснения (от кого документы, что это) -- учти при классификации.
3. Убедись, что `.vassal/case.yaml` существует и стороны определены. Если нет -- предложи `/vassal-litigator:init-case`.
4. Запусти `scripts/setup.sh` для установки OCR-зависимостей (если не запущен в этой сессии).
5. Выполни pipeline из SKILL.md: сохранение -> извлечение -> preview -> apply -> очистка.
6. Предложи экспресс-анализ аргументов оппонента (Фаза 4).
