# intake — Приём материалов клиента

Когда пользователь запускает эту команду с "$ARGUMENTS":

1. Прочитай скилл `skills/intake/SKILL.md` и следуй его инструкциям.
2. Если `$ARGUMENTS` содержит путь к папке — используй её как источник файлов.
3. Если аргументов нет — спроси, откуда взять файлы, или проверь текущую директорию.
4. Убедись, что `.vassal/case.yaml` существует. Если нет — предложи сначала `/vassal-litigator:init-case`.
5. Запусти `scripts/setup.sh` для установки OCR-зависимостей (если не запущен в этой сессии).
6. Выполни полный pipeline из SKILL.md: сохранение → извлечение → preview → apply.
